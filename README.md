Fluid dynamics environment (to provide learning data for NN)

# Quick start (OpenGL frontend):

### Mac OS:
```
brew install glfw3
```

### Ubuntu:

```
sudo apt-get install libglfw3-dev
```

```
make gl
./gl
```

<img src=./img/gl.gif width=500/>

## Or build the shared lib

`make sim.so`

and use through python

```
python3 sim.py
```
